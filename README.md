# Tabela-de-controle-de-investimentos-em-excel
Ferramenta de simulação de rendimentos de seus investimentos pessoais

O trabalho construido tem como objetivo principal a criação de uma  ferramenta automatizada para simulação de rendimentos de uma carteira financeira,
Criando fundamentos excenciais de excel em:

Formatação;

Uso de formulas;

Calculos e variáveis;

Utilização de planilhas auxiliares.

Uma ferramenta simples, com grande poder de auxilio, previsões e organização.
